# uraniborg Contents

## Major Directories

The following directories are created by the Git repository:

- `charts`: Star charts generated by `uraniborg` are saved here.
- `config`: Configuration files needed by `uraniborg` are kept here.
- `data`: Star catalogs used by `uraniborg` are kept here.
- `docs`: More detailed documentation of the various features and options in `uraniborg`.
- `examples`: Various sample charts for specific scenarios
- `fonts`: Several open-source TrueType fonts that can be used with `uraniborg`.


## Major files:

The following files are significant for `uraniborg` operation:

- `charts/output.png`: The default filename `uraniborg` uses when it saves a chart.
- `config/main.yaml`: The primary configuration file. It defines what a given chart should contain and look like. Changing this file automatically generates a new chart.
- `config/default.yaml`: A list of default values to use for chart configuration, when they have not been set elsewhere (e.g., in `config/main.yaml`).
- `config/application.yaml`: Application or "once per startup" configuration items. Unlike the ones in `config/main.yaml`, they do not change during the time `uraniborg` runs. Changes take effect when `uraniborg` is stopped and restarted.

More details about these files are given in the main README, as well as in the files under `docs`.
