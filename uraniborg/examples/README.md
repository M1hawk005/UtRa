# Additional Documentation

## Files in this directory:

- ["CONFIGURATION.md"](./CONFIGURATION.md): A walkthrough of how to configure a chart, with examples of configurations and the charts they produce.
- ["SAMPLES.md"](./SAMPLES.md): Additional sample charts generated by`uraniborg`.
- ["SCHEMES.md"](./SCHEMES.md): Sample charts for each of the schemes included with `uraniborg`.

## Subdirectories:

- ["introduction/"](./introduction): Contains the charts shown in ["CONFIGURATION.md"](./CONFIGURATION.md) and configurations used to create them.
- ["samples/"](./samples): Contains the charts shown in ["SAMPLES.md"](./SAMPLES.md) and ["SCHEMES.md"](./SCHEMES.md).
